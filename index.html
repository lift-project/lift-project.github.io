Lift

Where High-Level Programming Meets Performance Portability

Welcome to Lift

Lift is a novel approach to achieving performance portability on parallel accelerators.

Source Code

The Lift source code is available at https://github.com/lift-project/lift.
Lift is open source software released under the permissive MIT license.
